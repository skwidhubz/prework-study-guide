Git

  • GitBash terminal can communicate and link with GitHub
  • "code ." - Opens VSCode
  • "git status" - check the current location and status of where you are in GitBash
  • "git checkout" - change branch/location within git repository
  • "git add -A" - stage changes to repository
  • "git commit -m "note" " - commit changes to repository
  • " -m " - in the above command, this adds the message to the command
  • "cd" - change dir
  • "cd .." - exit dir to level above
  • "mkdir" - make dir
  • "ls" - list dir
  • "touch name.txt" - create text file
